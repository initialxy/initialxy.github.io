Hello World!

This is a minimalistic demo of fully modularized web components, which is being standardized into HTML5. When I say "fully modularized", I mean divided into as many files as possible. Currently this demo uses polymer to provide support for web components. Hopefully web components will get standardized enough such that they become entirely native. This demo incorporates the following web components features.

  • HTML template
  • Shadow DOM
  • Custom element
  • HTML import

You can download the source of this demo.
